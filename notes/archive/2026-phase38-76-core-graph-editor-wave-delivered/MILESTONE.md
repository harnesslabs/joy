- Core-first sequencing: graph/materialization and editor artifacts landed before package-manifest recursion

  - registry v2 embedded summaries
- `joy add` is now add + sync-lite by default to improve cargo/uv-like ergonomics
- Root `compile_commands.json` is auto-generated by default (copy refresh, not symlink)
- Lockfile and graph artifacts are first-class machine-readable state for diagnostics and automation
